# 3D Interactive Studio

A powerful, browser-based 3D scene editor built with React 18, Three.js, and Tailwind CSS.

## Features

- **Interactive 3D Scene**: Rotate, pan, and zoom around your creations.
- **Unified Menu**: All controls (Add Object, Properties, Settings) are now located in the top-left menu.
- **Transform Controls**: Select any object to Move, Rotate, or Scale it using visual handles.
- **Customization**: Change object colors, toggle auto-rotation, and switch between Light/Dark themes.
- **Responsive Design**: Works on desktop and mobile devices.

## How to Use

1. **Navigation**:
   - **Rotate**: Left-click and drag.
   - **Pan**: Right-click and drag.
   - **Zoom**: Scroll wheel.
2. **Editing**:
   - Click an object to select it.
   - Use the bottom toolbar to switch between Move, Rotate, and Scale modes.
   - **Open the Menu** (top-left hamburger icon) to:
     - Add new shapes (Cube, Sphere, etc.).
     - Change the color or position of the selected object.
     - Delete objects or clear the scene.
     - Change the app theme.
3. **Menu**:
   - Click the top-left hamburger icon to access all tools and settings.
